# Microsoft Sentinel Deployment

## Objective
Deploy Microsoft Sentinel and associate it with the HomeSOC Log Analytics Workspace.

## Actions Performed
- Created Azure Resource Group: HomeSOC-RG
- Created Log Analytics Workspace: HomeSOC-LAW
- Enabled Microsoft Sentinel on the workspace

## Result
Microsoft Sentinel was successfully deployed and connected to the HomeSOC Log Analytics Workspace.
