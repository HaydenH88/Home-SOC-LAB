# Azure Arc Onboarding

## Objective
Connect a Windows 11 endpoint to Azure Arc for centralized management and monitoring.

## Actions Performed
- Created Azure Arc onboarding script.
- Installed Azure Connected Machine Agent.
- Authenticated to Azure.
- Registered endpoint with Azure Arc.

## Result
The endpoint DESKTOP-MLQ9R3T successfully connected to Azure Arc and is visible within the HomeSOC-RG resource group.
